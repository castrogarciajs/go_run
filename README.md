# Go Cli

## Go_run 

- `Go_run` es CLI escrito en Go en donde podras listar, eliminar y completar tareas.

## ventajas

- Al estar escrito es Go su respuesta sera optima y precisa.

## usage

* Instalacion 

```go
go install https://github.com/sebastian01w/go_run
```

* comandos
````go
[
    "--posts",
    "--create",
    "--completed",
    "--delete"
]
````

**Example**

* *--posts*
````sh
go_run --posts

<list posts>
````

* *--delete*
````sh
go_run --delete <id>
````
